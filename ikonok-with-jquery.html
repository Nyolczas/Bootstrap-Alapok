A Bootstrap alapértelmezésben a Glyphiconokat (https://glyphicons.com) használja.

Szinte bárhova beilleszthetőek (szövegbe, gombokra, navigációs elemekre, űrlapmezőkbe...) a következő kódrészlettel:

span class="glyphicon glyphicon-NAME" /span

Ikonok

Fontos linkek a Bootstrap témához:

A Bootstrap Dokumentáció nagyon látványos és könnyen használható. Ha valahol elakadsz, érdemes itt nézelődni először.

A dokumentáció Components (komponensek) aloldalán megtalálod a Bootstrap összes felhasználható elemét, példákkal együtt.

A Bootsnipp egy remek oldal, ahol ingyenes Bootstrap kódrészleteket tudsz letölteni. Ezeket kevés módosítással könnyen be tudod építeni a saját projektjeidbe.

Végül a WrapBootstrap -en fizetős Bootstrap sablonokat vagy akár kész weboldalakat tudsz megvásárolni, illetve inspirálódhatsz is belőlük.
